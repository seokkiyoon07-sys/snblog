- 2024.07 학평 13번(교육청): 배점 4점, 난이도 중상, 정답률 0.51

현재 문항(10153)의 난이도·정답률 추정
- 난이도: 상
- 추정 정답률: 약 48% ~ 58% (대표값 약 52%)
근거: 본 문항은
1) 좌표 기하로 원 ω1의 교점 E, G를 정확히 선별(호 조건 포함)해야 하고,
2) 내적을 통한 각 계산 → 현 길이 EG 산출,
3) 삼각형 CEG에 헤론 공식 적용 → Ω의 외접반지름 R = abc/(4Δ),
4) 마지막에 GH = 2R sin∠HCG = 2R sin∠BAC로 귀결
이라는 4단계 구조입니다. 각 단계에 산술·대수와 기하 판단(호 선택, 부호·분기)이 모두 개입되어 실수 여지가 크며, 실제 유사형(2024.07 학평 13번) 정답률이 0.51인 점을 고려하면 상 난이도로 보는 것이 타당합니다.

간단 분석(풀이 포인트 요약)
- 좌표 배치: B=(0,0), A=(-3,0), C=(0,4). D는 AB를 2:1 내분 → D=(-1,0), AD=2 → ω1: 중심 A, 반지름 2.
- E: AC 위에서 AE=2 → E = A + (2/5)(C−A) = (−9/5, 8/5).
- G: ω1∩{중심 C, 반지름 CG=2√6}에서 호 EF(작은 호, D 제외)에 해당하는 교점 선택.
- EG: A를 중심으로 한 원 위에서 벡터 내적으로 cos∠EAG 계산 → 현 길이 공식 EG = 2R1 sin(∠EAG/2)로 구함(R1=2).
- RΩ: ΔCEG를 헤론 공식으로 구해 RΩ = abc/(4Δ).
- 최종: ∠HCG = ∠BAC, 삼각형 ABC의 sin∠BAC = 4/5이므로 GH = 2RΩ sin∠BAC.
- 오답 포인트: 호 조건에 맞는 G의 선택 오류, 내적/현 길이 계산에서 부호·분기 실수, 헤론 공식·외접반지름 계산의 산술 실수.

간단 분석(핵심 풀이 포인트)

- 구간별 h′(x)
  - x<−1: h′(x)=x^2+ax+a. 판별식 Δ=a(a−4).
    - a<4: 근 없음 → 항상 양 → 이 구간에서 극값 없음.
    - a=4: 중근(x=−2) → 부호 변화 없음 → 극값 없음.

  - x≥1: h′(x)=−x^2+(a+1)x−a=−(x−1)(x−a).
    - 0<a≤1: x=1에서 +→−로 한 번 변화 → 극값 1개.
    - 1<a≤4: (1,a)에서 양, x>a에서는 음 → x=a에서 +→−로 한 번 변화 → 극값 1개.

(SNarGo) 요청하신 유사 문항 3개를 SN RAG에서 검색해 인용합니다. 이번 검색은 주제(지수·로그) 유사 위주로 매칭되었고, 좌표·대칭·중점 조건이 결합된 "복합 구조"까지 일치하는 항은 직접 매칭되지 않았음을 미리 말씀드립니다.

- 고2 2014.06.12 학력평가(교육청) 2번
- 고3 2015.10.13 학력평가(교육청) 22번

현재 문항(제시된 문항 22)에 대한 난이도/정답률 추정
- 구조적 비교: 
  - 로그함수 y=log_{16}(8x+2)와 지수함수 y=4^{x-1}-1/2 동시 활용
  - 대칭 y=x → 좌표 교환 (a,b)→(b,a)
  - A′가 OB 위: 벡터 비례 (b,a)=λ(x_B,y_B)
  - 중점 M으로 B 좌표를 a,b로 표현
  - 지수식 구조 일치로 λ=1/2를 찾아내는 관찰(결정적)
- 위 유사 문항들은 주제는 같으나 계산형으로 매우 쉬운 편입니다. 본 문항은 좌표기하 조건(대칭·직선·중점)과 지수·로그의 구조 매칭을 결합한 단계적 사고가 필요하여 체감 난이도가 뚜렷이 높습니다.

- A∈곡선1: b=log_{16}(8a+2) ⇔ 16^b=8a+2 ⇔ 4^{2b}=8a+2.
- 중점 M(77/8,133/8) ⇒ x_B=77/4−a, y_B=133/4−b.
- A′=(b,a)가 OB 위 ⇒ (b,a)=λ(x_B,y_B) ⇒ x_B=b/λ, y_B=a/λ.

(SNarGo) 요청하신 '유사 문항 3개·메타 기반 추정·간단 분석'입니다. 먼저 보고드립니다: SN RAG로 유사 검색을 2회 수행했으나(로그 보관), 이번 유형(포물선-타원 결합, 초점·준선/둘레·넓이 조건)의 직접 매칭 사례를 확보하지 못했습니다. 따라서 유사 문항 제시는 보류하고, 본 문항의 난이도·정답률 추정과 핵심 분석을 제공하겠습니다. 필요하면 추가 키워드로 재탐색을 진행하겠습니다.

난이도/정답률 추정
- 구조 난도: 포물선 정의 y^2=4px, AF=AH로 넓이와 결합해 대수식 도출 → 타원 중심·장반축을 x,p로 표현, F=(p,0)를 정점으로 강제하는 p=h±a 조건 분기 → p 결정 후 x,y 산출 → HF^2 계산. 고난도 연립·분기 판단과 기하적 해석이 결합된 4점형.
- 추정: 난이도 상, 정답률 약 0.40±0.07 (대략 0.33~0.47 범위).

간단 분석 포인트
- 좌표 설정: A=(x,y) with x>0,y>0, H=(-p,y), F=(p,0). 포물선 y^2=4px, 포물선 정의로 AF=AH=x+p.
- 넓이 조건: 삼각형 AHB는 밑 AH=x+p, 높이 y 이므로 (1/2)(x+p)y=2p+12 → (x+p)y=4p+24. 이를 y^2=4px와 결합해 p x (x+p)^2 = 4(p+6)^2 도출.
- 둘레 조건과 타원: AH+AB+HB=p+27, AH=x+p → AB+HB=27−x=2a. A,H가 같은 y좌표이므로 타원 중심 x좌표 h=(x−p)/2, 장반축 a=(27−x)/2.
- F가 타원 위 정점: p=h±a를 대입해 분기. “+” 분기에서 p=9가 유일하게 타당. “−” 분기는 타원 내부성 위배로 배제.
- 수치 결정: p=9를 p x (x+p)^2 = 4(p+6)^2에 대입 → x(x+9)^2=100 → x=1(유일 실근), y^2=4px=36 → y=6.
- 최종: HF^2 = (2p)^2 + y^2 = 4p^2 + y^2 = 4·9^2 + 36 = 360 → k^2=360.
- 실수 포인트
  1) p=h±a 분기에서 부적합한 가지를 끝까지 끌고 가는 오류
  2) 넓이식 (x+p)y=4p+24와 y^2=4px의 결합에서 항 정리 실수
  3) 타원 중심 평균성(h=(x−p)/2)과 2a=27−x의 대응 누락
  4) HF^2 계산 시 (−p−p)^2=4p^2 항을 빠뜨리는 실수
