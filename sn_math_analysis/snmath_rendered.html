🔧 간단 분석 (현재 문제)

핵심 개념

  • 거듭제곱 적분:
    $$\int x^n dx = \frac{x^{n+1}}{n+1}$$
  • 적분의 선형성
  • 초기조건으로 C 결정

풀이 개략

  1. 부정적분 계산:
    $$F(x) = \int (4x^3 - 2x) dx = x^4 - x^2 + C$$
  2. 초기조건 적용:
    $$F(0) = 4 \Rightarrow C = 4$$
  3. 특정값 계산:
    $$F(2) = 2^4 - 2^2 + 4 = 16$$

📊 현재 문항의 난이도/정답률 추정

유형 적합성: 주어진 문제는 "닫힌구간 [-2,2]에서 연속인 다항함수의 |f(x)| 최댓값 = k의 최솟값"을 찾는 기본형입니다.

핵심은 $f'(x) = 6(x+2)(x-1)$에서 임계점 $x = -2, 1$과 구간 끝점 $x = -2, 2$에서의 함수값을 계산하고 절댓값으로 비교하는 표준 절차입니다.

추정 결과: 난이도 중하~중, 예상 정답률 약 0.88~0.93(대표값 0.90 내외), 배점 3점급

🔧 간단 분석 (현재 문제)

핵심 개념

  • 최댓값 정리: 폐구간에서 연속인 함수는 최대·최솟값을 가짐
  • 임계점과 끝점 비교: $f'(x) = 0$의 해와 구간 끝점에서 $f(x)$ 평가
  • 절댓값 처리: 최종적으로 $|f(x)|$ 값들 중 최대가 $k$의 최솟값

계산 포인트

  1. 임계점 찾기:
    $$f'(x) = 6(x+2)(x-1) \Rightarrow x = -2, 1$$
  2. 후보점에서 함수값 계산: $x = -2, 1, 2$에서 $f(x)$ 계산
    • $f(-2) = 12$
    • $f(1) = -15$
    • $f(2) = -4$
  3. 절댓값 비교:
    $$|12| = 12, \quad |-15| = 15, \quad |-4| = 4$$
    최대값은 15
  4. 결론: $k$의 최솟값은 15
